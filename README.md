# How to run server and client?

## Server

### Run the server
```
cd server
npx nodemon
```

## Client

### Compiles and hot-reloads for development
```
cd client
npm run serve
```

### Compiles and minifies for production
```
cd client
npm run build
```

### Lints and fixes files
```
cd client
npm run lint
```
